# 凡人界 MUD - 基建框架版

## 项目说明
基于《凡人界》开发规范的纯文本MUD游戏框架。
- **静态房间**：所有房间固定定义，不可动态生成
- **核心基建**：人物、背包、地图、商店、功能建筑、命令系统
- **关系身份**：青梅竹马、隔壁少妇等关系驱动
- **扩展性**：预留性爱系统、后宫、挖宝、灵宠等接口

## 快速开始
```bash
cd mud_game
python3 server.py
```

然后用 Telnet 连接：
```
telnet localhost 8888
```

## 已实现基建

- 房间静态固定
- 基建优先，内容后填充
